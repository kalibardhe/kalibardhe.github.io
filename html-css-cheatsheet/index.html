HTML / CSS
HTML
Tag Name Description

<h1></h1>

 Heading Section Heading element.
It goes from h1 to h6.

<a></a>

 Anchor tag The Anchor element.
Creates a hyperlink to web pages, or anything a URL can address.

<audio></audio>

 Audio element The embed Audio element.
It is used to embed sound content in documents.

<title></title>

 Title element The Document Title element.
It defines the document's title that is shown in a browser's title bar or a page's tab.
CSS
Property Name Description

padding

 padding Sets the padding area on all four sides of an element at once.

width

 width Sets an element's width.
By default, it sets the width of the content area.

flex

 flex Sets how a flex item will grow or shrink to
fit the space available in its flex container.

float

 float Places an element on the left or right side of its container,
allowing text and inline elements to wrap around it.
